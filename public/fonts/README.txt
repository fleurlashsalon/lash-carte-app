PDFの日本語を正しく表示するには、ここに次のいずれかのフォントを置いてください。
本文は 12pt で出力されます。

【推奨】Noto Sans JP（文字化け・「フォントが正しくない」を防ぎます）
1. https://fonts.google.com/noto/specimen/Noto+Sans+JP を開く
2. 「Download family」で ZIP をダウンロード
3. 解凍し、NotoSansJP-Regular.ttf をこのフォルダにコピー
- ファイル名は「NotoSansJP-Regular.ttf」にしてください（こちらを最優先で使用します）

【任意】メイリオ（Meiryo）で出力したい場合
- Windows の C:\Windows\Fonts\ から Meiryo.ttf をコピーしてこのフォルダに貼り付け
- ファイル名は「Meiryo.ttf」のまま
- 環境によっては「フォントが正しくない」と出る場合があります。その場合は上記 NotoSansJP-Regular.ttf を配置してください。

フォントがなくてもPDFは出力されますが、日本語が文字化けします。
